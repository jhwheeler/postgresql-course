# Sample answers

Solutions to the workbook exercises in the [curriculum](../README.md),
written by someone still learning Postgres — treat them as a starting
point, not a reference implementation. If you spot a bug, a more idiomatic
approach, or a missed edge case, please open an issue or PR. Corrections
very welcome :)

## Filenames

File names here match the bolded slugs in the [curriculum](../README.md)
(e.g. `row_counts_all_tables.sql` for exercise 1). If you save your own
solutions with the same names, diffing against these samples is trivial.
